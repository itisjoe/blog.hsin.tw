alias ll='ls -a'

設定後 輸入 ll 就會等於輸入 ls -a

輸入 alias 可以看目前有哪些指令別名

如果要讓這個指令別名永久有效
要寫入個別使用者的 ~/.bashrc 中

echo "alias ll='ls -a'" >> ~/.bashrc

或是直接編輯 ~/.bashrc 這個檔案 加入需要的指令別名

如果要即時啟用的話 接著輸入下面這行 (不然下次登入才會生效)

source ~/.bashrc
