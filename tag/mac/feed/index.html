以下指令都是輸入在終端機中

▼ 找出所有以 ._ 開頭的檔案

find /Volumes/檔案位置 -name "._*"

▼ 刪除所有以 ._ 開頭的檔案 即在後面加上 -delete

find /Volumes/檔案位置 -name "._*" -delete

像是如果要找名稱為 JetDrive32G 的隨身硬碟
即是如下

find /Volumes/JetDrive32G/ -name "._*"

]]> https://itisjoe.github.io/blog.hsin.tw/2015/mac-display-hidden-file-and-path/feed/ 0 [Mac] Parallels Desktop 7的一些注意事項 https://itisjoe.github.io/blog.hsin.tw/2011/mac-parallels-desktop-7-notice/ https://itisjoe.github.io/blog.hsin.tw/2011/mac-parallels-desktop-7-notice/#respond Wed, 28 Dec 2011 05:14:53 +0000 https://itisjoe.github.io/blog.hsin.tw/?p=491 用了幾個月的VirtualBox還是覺得不順

所以決定還是付錢買個Parallels Desktop(以下簡稱PD)吧
PD 7已經是多語言的了
所以可以下載安裝檔案裝
然後買簡體中文或港版的序號 => this way
買人民幣或港幣的都滿便宜的 不到NT1000
就看匯率囉

▼ 安裝win7完後 出現下面這個問題
FATAL ERROR: NO free memory for XSDT.
Press any key to continue booting.

[解決方法]
1. Go to Parallels Desktop Configuration --> Hardware --> Boot Order and find Boot Flags field:
2. Copy/paste the following line there:
kernel.waet.enable=0
3. Save the changes and start your virtual machine.
[ref.]
http://www.mobile01.com/topicdetail.php?f=480&t=2350771&last=30936707
http://kb.parallels.com/en/111406 (這邊有圖例)

▼ 雙螢幕時 將全螢幕的PD畫面拉去外接顯示器
1. 先到 Virtual Machine > Configure > Option > Full Screen
2. 將Use Mac OS X Full Screen 的勾勾拿掉

PS.
1. PD7 必須先昇級到 7.0.14922
2. Virtual machine 要先關機
[ref.]
http://www.mobile01.com/topicdetail.php?f=482&t=2354633&last=30909625

▼ Mac跟Win7共用資料夾
在Parallel視窗的右下角有一個看起來像是兩個齒輪的icon
必須要點選那個去下載Parallel Tools
待它自動安裝好後
就會在Parallel視窗的桌面跳出”Parallel Shared Folders”
這樣子就可以同時在Windows的環境看到原本儲存在Mac系統下的檔案了

Parallel Shared Folders的路徑是\\psf
預設打開的是Mac的home
打開的時候Mac這邊的Dock上會出現一個長得像Win檔案總管的東西
[ref.]
http://www.mobile01.com/topicdetail.php?f=482&t=2379925&r=3&p=1

▼ 與host的Mac用同一層網路
預設是共用網路 所以虛擬機器裡的內網只看得到自己
要改成看得到和host的Mac一樣的內網

[解決方法]
Focus到該虛擬機器
工具列上的”虛擬機器” -> 配置 -> 硬體 -> 網路 -> 類型
從”共用網路” 改選成”橋接網路”
[ref.]
http://www.oikos.com.tw/v4/viewtopic.php?id=45159

▼ windows下無法切換輸入法(ctrl+space,ctrl+空白鍵)
在win7虛擬機器裡 都未更動的情況下 如果按ctrl+space 會打開Mac下的spotlight
這樣就沒辦法切換輸入法

[解決方法]
PD的”偏好設定” -> 鍵盤 -> 啟用 Mac OS X 系統快速鍵
將這個勾勾取消
之後就不會打開Mac的spotlight 而是win裡的切換輸入法了
