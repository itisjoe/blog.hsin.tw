    firefox 擴充套件 (20090821更新)

    2009-02-09

    我有在用或是曾經有用過的firefox plugin

    這是我的收藏集
    下面說的大部分都有在裡面囉 看到喜歡的可以直接裝XD

    http://www.moztw.org/firefox/extension_top_15/
    這是第三屆Top15的列表 我好像幾乎都有裝喔XD
    不過還是加減列一下我有裝的

    IE Tab
    可以在fx中直接用ie的引擎開啟，要看IE Only的網站很方便

    FireGestures
    讓fx支援滑鼠手勢

    新同文堂
    可以讓網頁的文字繁簡互換 看大陸網站的時候很方便

    Adblock Plus
    可以把網頁中的廣告擋住不顯示 畫面就不會亂七八糟啦~

    Download Statusbar
    會把下載中的檔案跟進度顯示在瀏覽器下面 就不會擋住畫面了

    Foxmarks 書籤同步
    可以讓不同電腦上的fx書籤同步 這樣家裏跟公司的fx書籤可以一致 相當之好用阿

    Video DownloadHelper
    我主要是拿來抓youtube的flv檔 不用設定什麼 他自己會顯示有什麼檔可以下載 很方便！

    Menu Editor
    整理每個選單中要顯示的項目 右鍵選單不會再又臭又長啦～

    Greasemonkey
    可以自訂使用者script 對特定的網頁再作處理 這真的太好用啦

    Personal Menu
    可以整理選單要出現的選項 通通隱藏只留下一個按鈕 再把有需要的選項放進去就好 我喜歡越簡潔越好阿~

    Firebug
    檢視網頁的內容 偵錯網頁

    Favicon Picker
    把書籤的icon換掉 我拿來換bookmarklet的圖示~

    autoHideStatusbar
    隱藏下面的狀態列 但是可以設定移到連結或是某些網頁時 可以顯示狀態列

    Easy DragToGo
    用滑鼠對連結、文字或是圖片作拖曳 來達成某些功能 直接另存圖片或是咕狗文字
    這個超懶人的 跟滑鼠手勢一樣懶XD

    Fission
    將進度列移到網址列的背景來執行
    也可以設定在網址列顯示滑鼠所指的超連結網址
    這個跟autoHideStatusbar搭配使用 大部分時間 就可以不用顯示狀態列囉

    Smart Stop/Reload
    之前講的Stop-or-Reload在fx3.5已經不支援囉
    所以找到這個相同功能的addon

    Tab Mix Plus
    增強firefox的分頁功能

    Omnibar
    智慧型網址列 加強網址列的功能 可以設定在網址列輸入文字後要使用的搜尋引擎
    還可以設定不同的關鍵字來在不同的搜尋引擎搜尋

    Add to Search Bar
    將任何一個網頁的搜尋列 加到firefox裡的搜尋列中 之後就可以直接搜尋囉

    Stylish
    修改頁面的css

    PlacesCleaner 收藏庫清潔工 new
    清理收藏庫 刪除瀏覽歷史中 瀏覽次數少於兩次的資料

    =======================
    下面是已經沒有在用的囉
    =======================

    keyconfig
    自訂firefox的熱鍵
    按這裡直接安裝

    Stop-or-Reload
    把重新載入跟停止的按鈕合在一起 越簡潔越好~~~

    Tab Mix Lite CE
    是一個類似Tab Mix Plus的分頁擴充套件
    note:這個安裝網址是在大陸的網站，如果有疑慮的人請不要安裝

    Sxipper
    可以自動填寫表單的addon跟管理多人的帳號密碼 我是拿來填每天只能填一次的抽獎表單XD 省很多時間
    這是咕狗到的->[FF] 方便好用的網頁表單填表工具 -- Sxipper
