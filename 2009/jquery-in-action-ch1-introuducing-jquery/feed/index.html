focus()參考這頁
http://api.jquery.com/focus/
直接對要操作的input加上一個id 例如是myid1
$(“#myid1”).focus();

readonly跟disabled都是更改屬性就好
https://itisjoe.github.io/blog.hsin.tw/2012/jquery-easy-steps/
$(“#myid1”).attr(“disabled”,”disabled”); //設為disabled
$(“#myid1”).attr(“disabled”,””); //取消disabled

$(“#myid1”).attr(“readonly”,true); //設為readonly
$(“#myid1”).attr(“readonly”,false); //取消readonly

$(“#form1 input[type=checkbox]:checked”).each(function(){
alert($(this).val());
});

全選、全取消、反向選擇 都也是改屬性而已
全選
$(“#form1 input[type=checkbox]”).attr(“checked”,true);
全取消
$(“#form1 input[type=checkbox]”).attr(“checked”,false);
反向選擇
$(“#form1 input[type=checkbox]”).each(function(){
if($(this).attr(“checked”)==true){
$(this).attr(“checked”,false);
} else {
$(this).attr(“checked”,true);
}
});

radio的checked跟上面類似
alert($(“#form1 input[type=radio]:checked”).val());

改文字顏色跟背景顏色
https://itisjoe.github.io/blog.hsin.tw/2012/jquery-easy-steps/
這篇都有寫

改標籤裡的文字 是html()
$(“div#content”).html(“這是更新的文字”);

onclick要改用
$(“#btn1″).click(function(){
// 要做的事
});

以最後一個為例 更改自己的名字及顏色
先給這個input加一個id (每個標籤的id都要不同 一個網頁上 一個id只能出現一次)
<input type=”button” id=”btn” value=”替換為自己的名字,字顏色為藍色粗體” />
在script標籤中
則是寫
$(“#btn”).click(function(){
$(“#htmlText”).html(“Lan”);
$(“#htmlText”).css(“color”,”#0000ff”);
$(“#htmlText”).css(“fontWeight”,”bold”);
});
其他都一樣
